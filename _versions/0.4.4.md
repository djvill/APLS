This is a "preview" version of APLS.
We have uploaded **all remaining transcript series**; however, we are still tweaking transcript data and refining the UI, so the data is subject to change in minor ways.


#### New layers/attributes
{:.no_toc}

- Added new <span class="layer">coronal_stop_deletion</span> layer, the first layer in the `user_contrib` [project]({{ '/doc/view-transcript#projects' | relative_url }}). This layer represents Dan Villarreal's hand-codes for 14,158 tokens of coronal stop deletion (see [ADS 2026 slides](https://github.com/djvill/djvill.github.io/blob/main/research/presentations/VillarrealRechsteiner2026_ADS_CSD-APLS.pdf) for more information).
- Added new <span class="layer">hesitation</span> layer, which encodes whether or not the word was terminated prematurely. This is the same information that's represented by the presence/absence of the [hesitation marker `~`]({{ '/doc/layer-typology#layers-by-notation-system' | relative_url }}) in the <span class="layer">word</span> or <span class="layer">orthography</span> layers, but putting this information in its own layer is more transparent to users and can make searching easier.
